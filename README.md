# 🤖 Voice Assistant with GUI

A Python-based desktop voice assistant with a clean modern GUI built using Tkinter. It listens to user commands, processes them using a custom knowledge base, and responds via text and text-to-speech.

## ✨ Features
* 🎤 **Voice Recognition:** Uses Google Speech Recognition API.
* 🔊 **Text-to-Speech:** Provides instant audio feedback using `pyttsx3`.
* 💻 **Modern UI:** Built with Tkinter featuring a dark theme.
* 🌐 **Web Operations:** Can search Google and open YouTube.
* 🕒 **Time & Date:** Tells current time and date.
